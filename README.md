# User Authentication System (Flask)

## Features
- User Registration
- Login System
- Password Hashing
- Session Management
- Logout

## Technologies Used
- Python (Flask)
- HTML, CSS
- SQLite

## Authentication Flow
1. User registers
2. Password is hashed and stored
3. User logs in
4. Session is created
5. Dashboard is accessed
6. User logs out

## Run Project
pip install flask werkzeug
python app.py
